# Smart Industrial Monitoring System

An Arduino-based industrial safety monitoring system that uses temperature sensor & gas sensor, to detect unsafe conditions and automatically trigger alerts and motor shutdown.

## Features
- Temperature monitoring
- Gas leakage detection
- Automatic buzzer alert
- Motor protection system
-Real-time safety monitoring

## Components Used
Gas Sensor
,Red LED
,Green LED
,Piezo
,Arduino Uno R3
,1kΩ Resistor
,Temperature Sensor[TMP36]
,DC Motor

## Working
The system continuously monitors industrial conditions using multiple sensors. If abnormal temperature, gas leakage the Arduino activates a buzzer alarm, turns on a warning LED, and stops the motor to prevent accidents or equipment damage.Under normal conditions, the motor runs safely and the system indicates normal operation.
This project demonstrates sensor interfacing, automation, and real-time safety monitoring using Arduino
